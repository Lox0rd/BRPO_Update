## Описание проекта

Программа объединяет три отдельных модуля (`programm_1`, `programm_2`, `programm_3`), разработанных разными участниками, в единый исполняемый файл. Для доступа к функционалу программы пользователь должен пройти аутентификацию — ввести корректный логин и пароль.

**Основные компоненты:**
* `main.c` — главный файл программы, содержит логику аутентификации и управление выполнением модулей.
* `programm_1(Lev).c` — модуль № 1 (разработан участником Lev).
* `programm_2.c` — модуль № 2.
* `programm_3.c` — модуль № 3.
* Папка `code/` — содержит все исходные файлы `.c`.

## Функционал

После успешной аутентификации пользователь может выбрать один из трёх модулей для выполнения. Каждый модуль реализует свою часть общего задания РГЗ.

## Требования

* Компилятор **GCC** (GNU Compiler Collection).
* Операционная система: Linux, macOS или Windows (с поддержкой GCC, например, через WSL, MinGW или Cygwin).

**!Необходимо создать paslog.txt файл и прописать в нём логин в первой строка, а пароль во второй!**

Команда для компиляции (**__gcc code/main.c code/programm_1(Lev).c code/programm_2.c code/programm_3.c -o programm__**)
